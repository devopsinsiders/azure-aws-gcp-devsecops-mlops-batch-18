# 🧪 **Assignment: Terraform Setup with Single `.tf` File**

## 🎯 Objective

By the end of this assignment, you will:

* Create folder structure in your system
* Install and use VS Code
* Create and manage files
* Open editor and terminal
* Write and execute Terraform using **only one file (`main.tf`)**

---

# 🧱 **Part 1: Create Folder in System**

## 🔹 Step 1: Create Root Folder

1. Open **File Explorer**
2. Go to `C:\`
3. Right click → **New → Folder**
4. Name it:

```
DevOpsInsiders
```

---

## 🔹 Step 2: Create Terraform Folder

Inside `DevOpsInsiders`, create:

```
Terraform
```

---

## 🔹 Step 3: Create Project Folder

Inside `Terraform`, create:

```
project1
```

---

## 🔹 Final Path

```
C:\DevOpsInsiders\Terraform\project1
```

---

# 📄 **Part 2: Create Only ONE `.tf` File**

## 🔹 Step 4: Create File

Inside `project1`:

1. Right click → **New → Text Document**
2. Rename it to:

```
main.tf
```

⚠️ Important:

* Extension must be `.tf`
* Not `.txt`

---

# 💻 **Part 3: Install VS Code**

## 🔹 Step 5: Install VS Code

1. Go to: [https://code.visualstudio.com/](https://code.visualstudio.com/)
2. Download and install
3. Keep default settings

---

# 📂 **Part 4: Open Folder in VS Code**

## 🔹 Step 6: Open Project

1. Open VS Code
2. Click **File → Open Folder**
3. Select:

```
C:\DevOpsInsiders\Terraform\project1
```

---

# 📁 **Part 5: VS Code Basics**

## 🔹 Step 7: Explorer

* Left side = **Explorer**
* Shows your `main.tf`

---

## 🔹 Step 8: Open Editor

* Click `main.tf`
* It opens in **Editor (center screen)**

---

# 📝 **Part 6: Write Terraform Code**

## 🔹 Step 9: Paste Code in `main.tf`

```hcl
resource "azurerm_resource_group" "example" {
  name     = "my-first-rg"
  location = "East US"
}
```

---

# 🖥️ **Part 7: Open Terminal**

## 🔹 Step 10: Open Terminal

### Method 1:

* Click **Terminal → New Terminal**

### Method 2:

```
Ctrl + `
```

---

## 🔹 Step 11: Check Path

Run:

```cmd
cd
```

👉 Output should be:

```
C:\DevOpsInsiders\Terraform\project1
```

---

# ⚙️ **Part 8: Install Terraform**

## 🔹 Step 12: Install Terraform

1. Go to:
   [https://developer.hashicorp.com/terraform/downloads](https://developer.hashicorp.com/terraform/downloads)
2. Download ZIP
3. Extract
4. Add to **Environment Variables**

---

## 🔹 Step 13: Verify Installation

```bash
terraform --version
```

---

# 🚀 **Part 9: Run Terraform**

## 🔹 Step 14: Initialize

```bash
terraform init
```

---

## 🔹 Step 15: Validate

```bash
terraform validate
```

---

## 🔹 Step 16: Plan

```bash
terraform plan
```

---

## 🔹 Step 17: Apply

```bash
terraform apply
```

Type:

```
yes
```

---
